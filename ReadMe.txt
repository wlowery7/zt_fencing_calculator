Thank your for downloading Zorro Technologies’ ™ Fencing Cost Calculator!

This program utilizes developer tools BeautifulSoup4 and Selenium
to fetch live pricing data from the Lowe's Hardware Store nearest to the user.

This application was packaged using cx_freeze.

To run this program, the user must already have Google's Chrome browser installed.

*******************************************************************************
In order to verify your PC meets the requirements to run this program,
please follow the instructions below.

1. In the folder labeled "resources," Launch the application chromedriver.exe, 
   then close the application.
 
2. Open the .txt file "uChromedriverPath" and follow the 
   instrcuctions provided.
